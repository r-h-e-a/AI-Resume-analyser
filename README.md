# AI-Resume-analyser


## Overview

The AI Resume Scanner is a web-based application designed to analyze and evaluate resumes based on predefined criteria. The system enables users to upload resumes in PDF or DOC format and generates a score by processing the content using rule-based logic. The application is built using a full-stack architecture with a React frontend, Node.js backend, and MongoDB database.

---

## Features

* Resume upload functionality (PDF/DOC formats)
* Automated content extraction and analysis
* Rule-based scoring mechanism
* User-friendly interface
* Backend processing using Node.js and Express
* Data storage using MongoDB

2. The system processes and analyzes the content.
3. A score or evaluation result is generated based on predefined rules.
